🎯 Демонстрация модального окна прогресса

📖 Как использовать

Модальное окно прогресса предоставляет следующие функции:

  • showProgressModal(title, cancellable) - показать окно
  • updateProgress(percent, message) - обновить прогресс
  • setProgressStatus(status, message) - установить статус (complete/error)
  • closeProgressModal() - закрыть окно
  • setOnProgressCancel(callback) - обработчик отмены

Пример кода:

// Показать окно прогресса
showProgressModal('Загрузка файла', true);

// Установить обработчик отмены
setOnProgressCancel(() => {
    console.log('Пользователь отменил операцию');
    closeProgressModal();
});

// Обновлять прогресс
let progress = 0;
const interval = setInterval(() => {
    progress += 10;
    updateProgress(progress, `Загружено ${progress}%`);
    
    if (progress >= 100) {
        clearInterval(interval);
        setProgressStatus('complete', 'Загрузка завершена!');
        setTimeout(closeProgressModal, 1000);
    }
}, 500);
